# 🌙 AURA — منصة الطاقة والروحانيات

منصّة طاقة وروحانيات (عربي + إنجليزي)، مجانية الدخول، تجذب الناس بمحتوى يومي
ممتع وتحوّلهم لعملاء للكورسات وجلسات الأساتذة. تعمل كموقع ويب **وكتطبيق موبايل (PWA)**
قابل للتثبيت — كل ذلك بدون أي خادم، مباشرة على GitHub Pages.

> AURA — an Arabic + English energy & spirituality platform. Free to enter,
> draws users with fun daily content, and converts them into customers for
> courses and live sessions. Works as a website **and an installable mobile app (PWA)**.

---

## ✨ المميزات

- 🔮 **طاقة اليوم** — كشف يومي (رسالة، رقم الحظ، لون اليوم، العنصر) مع **عدّاد أيام متتالية**.
- ♈ **طاقة مخصّصة حسب البرج** — أدخلي تاريخ ميلادك (يُحفظ محلياً) لرسالة برج يومية.
- 🃏 **كروت الطاقة** — اسحب كرتاً واقرأ تفسيره، مع إعادة الخلط.
- 🔗 **مشاركة** — زر مشاركة للطاقة والكروت (Web Share + نسخ احتياطي).
- 📚 **الكورسات** — عرض الكورسات بأسعارها وزر اشتراك.
- 👩‍🏫 **الحجوزات** — نموذج حجز حقيقي (اسم، تواصل، تاريخ، وقت) يعمل عبر واتساب فوراً
  وجاهز للحفظ في **Supabase** عند تفعيله.
- ⭐ **AURA Premium** — قسم تشويقي لنسخة بدون إعلانات (مرحلة لاحقة).
- 📢 **أماكن إعلانات** — جاهزة لتفعيل Google AdSense من `config.js`.
- 💬 **الدعم والدفع** — PayPal + واتساب + شام كاش (قريباً) + دعم فني.
- 🌐 **عربي / إنجليزي** — زر تبديل كامل مع اتجاه RTL/LTR.
- 📲 **PWA** — قابل للتثبيت على الموبايل، يعمل بدون إنترنت، أيقونة خاصة.

## 🔌 التفعيل الاختياري (في `config.js`)

| الميزة | الحقل في config.js | ملاحظات |
|--------|-------------------|---------|
| الحجز + لوحة التحكم | `supabase: { url, anonKey }` | شغّلي `supabase-schema.sql` أولاً. بدونه يعمل الحجز عبر واتساب فقط. |
| الدفع المباشر | `paypalClientId` + `currency` | Client ID من PayPal Developer. بدونه يُستخدم رابط `paypal` (paypal.me). |
| الإعلانات | `adsense: { client, slotTop, slotBottom }` | بعد قبول حسابك في AdSense. بدونه لا تظهر إعلانات. |
| رابط الموقع | `site` | يُستخدم في روابط المشاركة. غيّريه عند استخدام نطاق خاص. |
| أوقات الجلسات | `timeSlots` | عدّلي الأوقات المتاحة للحجز. |

## 🛠️ لوحة التحكم الإدارية (`admin.html`)

تتيح إدارة الكورسات والأساتذة والحجوزات **بدون لمس الكود**. للتفعيل:

1. أنشئي مشروع Supabase وشغّلي `supabase-schema.sql`، وضعي `url` + `anonKey` في `config.js`.
2. في Supabase → **Authentication → Users → Add user** أنشئي حساب الأدمن (إيميل + كلمة مرور).
3. افتحي `…/admin.html`، سجّلي الدخول، وابدئي بإضافة/تعديل المحتوى.

> عند تفعيل Supabase، الموقع العام يعرض الكورسات والأساتذة من قاعدة البيانات تلقائياً
> (بدلاً من القيم في `config.js`)، فأي تعديل من اللوحة يظهر فوراً للزوار.

## 🔎 جاهزية الإطلاق والمشاركة

- بطاقة معاينة احترافية (Open Graph + Twitter) عند مشاركة الرابط — الصورة `icons/og.png`.
- `robots.txt` و `sitemap.xml` لمحركات البحث.
- اختصارات تطبيق (طاقة اليوم / الكروت / الحجز) من أيقونة التطبيق على الموبايل.

> ⚠️ روابط المشاركة تشير إلى `https://souad-ch.github.io/auraapp/`. إذا اختلف اسم
> المستودع أو استخدمتِ نطاقاً خاصاً، عدّلي الروابط في أعلى `index.html` و `config.js`
> و `sitemap.xml` و `robots.txt`.

## ✏️ كيف تعدّلين بياناتك الحقيقية

كل المحتوى القابل للتغيير موجود في ملف واحد فقط: **`config.js`**

1. `whatsapp` — رقم واتساب بصيغة دولية بدون `+` أو أصفار (مثال: `9639XXXXXXXX`).
2. `paypal` — رابط PayPal الخاص بك (paypal.me أو رابط دفع).
3. `courses` — عدّلي عناوين الكورسات، الوصف، والأسعار (عربي + إنجليزي).
4. `masters` — عدّلي أسماء الأساتذة وتخصصاتهم وخدماتهم.

النصوص العامة للواجهة (الترجمة) موجودة في `i18n.js` إن أردتِ تعديلها.

## 🚀 النشر على GitHub Pages

1. ادفعي الملفات إلى المستودع (تم ذلك تلقائياً عبر هذه الجلسة).
2. في GitHub: **Settings → Pages**.
3. تحت **Source** اختاري الفرع (مثلاً `main`) والمجلد `/ (root)` ثم **Save**.
4. بعد دقيقة سيظهر رابط مثل: `https://souad-ch.github.io/auraapp/`
5. افتحي الرابط على الموبايل → سيظهر خيار «تثبيت التطبيق».

## 🗂️ بنية الملفات

| الملف | الوظيفة |
|------|---------|
| `index.html` | هيكل الصفحة |
| `styles.css` | التصميم والأجواء السحرية |
| `config.js` | **بياناتك** (واتساب، PayPal، الكورسات، الأساتذة، Supabase، AdSense) |
| `i18n.js` | الترجمات + محتوى الطاقة والكروت والأبراج |
| `app.js` | منطق التطبيق |
| `admin.html` + `admin.js` + `admin.css` | لوحة التحكم الإدارية |
| `supabase-schema.sql` | جداول الحجوزات والكورسات والأساتذة (شغّليه في Supabase) |
| `robots.txt` + `sitemap.xml` | تحسين محركات البحث |
| `manifest.json` + `sw.js` | إعدادات تطبيق الموبايل (PWA) |
| `icons/` | أيقونات التطبيق + صورة المشاركة `og.png` |
